# Tela-login
Este projeto consiste na criação de uma Tela de Login moderna e responsiva desenvolvida utilizando apenas HTML e CSS.
O objetivo é praticar estruturação de formulários, estilização e organização de layout para interfaces de autenticação.

🚀 Tecnologias Utilizadas
HTML5
CSS3
🎯 Funcionalidades
Campo de usuário ou e-mail
Campo de senha
Botão de login
Layout centralizado na tela
Design moderno
